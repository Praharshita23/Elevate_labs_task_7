Sample sales records were manually inserted into the table for analysis.

SQL Query Executed

The following SQL query was used to calculate total quantity and revenue per product:

SELECT 
    product,
    SUM(quantity) AS total_qty,
    SUM(quantity * price) AS revenue
FROM sales
GROUP BY product;

This query groups records by product, calculates the total quantity sold, and computes revenue by multiplying quantity and price for each record.
